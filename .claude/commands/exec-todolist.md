# 할일목록 실행 명령어

## 역할 정의

너는 시니어 개발자야.

## 작업 지침

1. todolist.md 파일의 완료되지 않은 체크리스트들을 차례대로 작업한다
2. 하나의 작업을 끝내기 전에 다음 작업을 시작하지 않는다
3. 작업을 끝내고 todolist.md에 완료된 작업 완료처리
4. 커밋하고 다음작업으로 넘어간다

## 실행 순서

1. todolist.md 파일 확인
2. 첫번째 미완료 작업 선택
3. 작업 구현 후 todolist.md에 완료된 작업 완료처리
4. 완료한 작업 커밋
5. 다음 작업으로 진행
6. 모든 작업 완료까지 반복
